Sinatra - Starting

Apr 1st, 2014

Best way to learn? Building.

Buiding a simple application with sinatra and deploy in heroku to remember what is current year :).

$ mkdir currentyear
$ cd currentyear

Bundler

Why? Why not?

$ gem install bundler

Gemfile

$ vi Gemfile

$ bundle install

Coding

$ vi web.rb

Runing

$ ruby web.rb
# access localhost:4567

ERB

$ vi web.rb

$ mkdir views
$ vi views/index.erb

Heroku

$ vi Procfile

$ git add -A
$ git commit -a -m "..."
$ heroku create <APP_NAME>
$ git push heroku master

Project
Heroku: http://currentyear.herokuapp.com
Github: https://github.com/nicolaslazartekaqui/currentyear
