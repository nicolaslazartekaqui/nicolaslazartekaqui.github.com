Postgres - How to Work With Dump

Mar 21st, 2013

SQL

Exporting

one base

$ pg_dump -U username dbname -f dumpfile.sql

all bases

$ pg_dumpall -U username > dumpfile.sql

Importing

one base

$ psql -d dbname -U username -f dumpfile.sql

all bases

$ psql -U username -f dumpfile.sql
